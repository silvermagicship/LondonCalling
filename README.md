# LondonCalling

Static share page for the London accommodation report.

- Entry page: `index.html`
- Travel dates: 2026-08-21 to 2026-08-25
- Party: 3 adults, 1 room/apartment
- Data sources: Expedia, Booking.com, hotel official pages, Hotels.com, Trip.com, Airbnb, Google Maps

This folder is intentionally static so it can be published directly from the existing `ByteArts` repository with GitHub Pages or any static file host.
